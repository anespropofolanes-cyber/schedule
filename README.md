# 麻瓜專用每日卡片套版產生器

把「金句」或「英文單字」做成正方形圖片，放進 Apple Watch 的照片錶面輪流顯示。單一 HTML 檔、純前端、不需安裝與註冊，用瀏覽器打開即可使用。

這是一個個人自用的小工具，主要設計考量是：操作直覺、版面乾淨、不依賴任何後端或金鑰。

## 功能

- 兩種模式：每日金句、每日英文單字，內容各自獨立。
- 三種輸入方式：逐筆快速新增、批次貼上、從檔案匯入（`.txt` / `.json`）。
- 內建提示詞按鈕：一鍵複製「金句」或「英文單字」的整理提示詞，貼到 Claude 或 ChatGPT 聊天視窗（搭配文章截圖或網址）即可生出可匯入的內容。
- 英文單字卡：自動分頁，每張預設三個單字，可切 1 至 4；顯示單字、音標（DM Mono）、中文釋義，例句可選。
- 朗讀發音：以瀏覽器內建語音念出當前卡片的英文單字，可放慢語速（單字模式）。
- 版面：六組低彩度配色或自訂顏色，宋體／黑體切換，文字自動縮放置於中央安全區。
- 自動保存：內容與設定存於瀏覽器 localStorage，關閉後再開仍保留。
- 輸出：單張或整批打包成 ZIP（`.png`），尺寸預設 1088×1088。

## 使用方式

1. 用瀏覽器打開 `index.html`（或將檔案點兩下開啟）。
2. 上方選擇「每日金句」或「每日英文單字」。
3. 加入內容（逐筆新增、批次貼上，或從檔案匯入）。
4. 右側調整配色、字體與裝飾，預覽即時更新。
5. 下載圖片，存進 iPhone 專屬相簿，於 Watch App 設定照片錶面同步該相簿。

完整的逐步說明請見 `使用說明.md`（麻瓜版手冊）。

## 用聊天視窗生成內容

工具下方「需要內容？取得提示詞」提供兩顆按鈕：

- 金句提示詞
- 英文單字提示詞

點一下複製提示詞，貼到你的 Claude 或 ChatGPT 聊天視窗，連同文章截圖或網址送出；它會輸出可直接貼回「批次貼上」的內容。此流程使用既有的聊天訂閱，不需要 API 金鑰，工具本身也不含任何 API 呼叫。

## 部署（選用）

本工具為靜態檔案，可直接本機開啟，無需部署。若要放上 GitHub Pages：

1. 將主檔命名為 `index.html` 並推上倉庫。
2. 在 Repository 的 Settings → Pages 選擇來源分支，啟用後即可透過網址開啟。

## 隱私與安全

- 純前端、單一檔案，無後端、無追蹤、無 API 金鑰。
- 所有內容與設定只存在使用者自己的瀏覽器（localStorage），不會上傳。
- 程式碼中不含任何金鑰或秘密，可安心保存。
- 若內容屬個人自用、不希望公開，建議將倉庫設為 Private，或不啟用公開的 GitHub Pages。

## 技術說明

- 單一 HTML 檔，內含 CSS 與 JavaScript，無建置流程。
- 字型透過 Google Fonts 載入（Noto Sans TC、Noto Serif TC、DM Mono、DM Serif Display）。
- 打包 ZIP 使用 JSZip（CDN 載入）。
- 圖片以 Canvas 算繪。首次開啟與打包下載需連網，其餘操作在本機進行。
